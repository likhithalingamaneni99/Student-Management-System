# Student Management System

## Overview
Student Management System is a Python and MySQL based application used to manage student records efficiently.

## Features
- Add Student
- View Student Records
- Search Student by ID
- Update Student Details
- Delete Student Records

## Technologies Used
- Python
- MySQL
- mysql-connector-python
